# 记忆配对游戏 - Scratch 项目设计

## 📋 项目信息

- **项目名称**: 记忆配对
- **适合年龄**: 6-10岁
- **游戏类型**: 益智休闲
- **预计开发时间**: 1-2小时

---

## 🎮 游戏玩法

1. 12张卡片正面朝下，均匀排列成4行3列
2. 玩家点击一张卡片，它会翻面显示图案
3. 再点击第二张卡片，也翻面显示
4. 如果两张图案相同 → 保持翻开，得分+10
5. 如果两张图案不同 → 等待1秒后自动翻回去
6. 找到所有配对后，显示"你赢了！"和用时

| 背景 | 1个 | 游戏背景 |
| 计时器 | 1个 | 显示用时 |
| 得分 | 1个 | 显示分数 |
| 胜利文字 | 1个 | 游戏胜利时显示 |

---

## 📦 变量设计

| 变量名 | 类型 | 用途 |
|-------|------|------|
| 得分 | 变量 | 记录配对成功的次数 |
| 用时 | 变量 | 记录游戏开始后的秒数 |
| 第一张 | 变量 | 记录第一次点击的卡片编号 |
| 第二张 | 变量 | 记录第二次点击的卡片编号 |
| 配对数 | 变量 | 记录已配对成功的对数 |
| 游戏状态 | 变量 | 0=游戏中, 1=翻开第一张, 2=翻开第二张 |

## 🔧 核心逻辑流程

### 主程序流程图

```
开始
  ↓
初始化变量
  ↓
随机打乱卡片顺序
  ↓
显示所有卡片背面
  ↓
开始计时
  ↓
┌─ 等待点击 ─┐
  ↓
玩家点击卡片
  ↓
判断游戏状态
  ↓
┌─────┴─────┐
↓           ↓
状态=0    状态≠0
  ↓           ↓
翻看第一张   翻看第二张
  ↓           ↓
设置状态=1   判断两张是否相同
  ↓              ↓
┌─────┴─────┐   ↓
↓           ↓   ↓
相同       不同
  ↓           ↓
得分+10    等待1秒后翻回去
配对数+1   状态=0
  ↓           ↓
检查是否全部配对  ←────┘
  ↓
┌─────┴─────┐
↓           ↓
全部配对   未完成
  ↓           ↓
显示胜利   返回等待点击
  ↓
游戏结束
```

### 关键积木逻辑

#### 1. 卡片初始化
```
当绿旗被点击
初始化所有变量
为每张卡片分配随机图案
显示卡片位置
```

#### 2. 点击卡片处理
```
当角色被点击
如果 游戏状态 = 0 且 这张牌未翻开
    翻牌（显示正面）
    如果 第一张 = 0
        设置 第一张 = 卡片编号
        游戏状态 = 1
    否则
        设置 第二张 = 卡片编号
        游戏状态 = 2
        等待 0.5 秒
        判断配对
```

#### 3. 判断配对
```
如果 第一张的图案 = 第二张的图案
    得分 = 得分 + 10
    配对数 = 配对数 + 1
    第一张 = 0
    第二张 = 0
    游戏状态 = 0
    如果 配对数 = 6
        显示胜利画面
否则
    等待 1 秒
    把第一张翻回背面
    把第二张翻回背面
    第一张 = 0
    第二张 = 0
    游戏状态 = 0
```
